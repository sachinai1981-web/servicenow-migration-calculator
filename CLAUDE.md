# ServiceNow Migration Cost Calculator

Free tool for enterprise CIOs to estimate migration cost from ServiceNow to Freshservice.

## Files
- index.html — HTML structure and CSS styling (all inline)
- calculator.js — All calculation logic and email gate
- README.md — Project docs

## How to run
Open index.html directly in browser — no build step needed.

## Deployment
GitHub Pages: https://sachinai1981-web.github.io/servicenow-migration-calculator
Push to main → auto-deploys via GitHub Actions

## Purpose
Top-of-funnel tool for Freshworks sales. CIOs calculate migration cost, enter work email to see results, get captured as warm leads for Freshworks demo booking.
